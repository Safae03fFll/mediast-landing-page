## Explication de l’étape 5

L’étape 5 se trouve dans la section `🔗 Connexion Google Sheets` du `README.md`.

### Ce que tu dois faire

1. Ouvre ton Google Sheet créé à l’étape 1.
2. Rafraîchis la page du navigateur.
3. Tu dois voir apparaître un nouveau menu dans la barre de menu de Google Sheets :
   - `🌾 RiceSkin`

### Ensuite

- Clique sur `🌾 RiceSkin`
- Puis sélectionne `Initialiser les en-têtes`

Cela va ajouter les colonnes nécessaires pour recevoir les commandes envoyées depuis l’application.

> En résumé : le menu `🌾 RiceSkin` se trouve dans ton Google Sheet, pas dans l’application React.
